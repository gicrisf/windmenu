# Supported Keys

Valid key names for `keys = [...]` command combinations in `windmenu.toml`:

## Modifier Keys

- `ALT` — Alt key
- `CTRL` or `CONTROL` — Control key
- `SHIFT` — Shift key
- `WIN` or `WINDOWS` — Windows key

## Special Keys

- `TAB`
- `ESC` or `ESCAPE`
- `SPACE`
- `ENTER`

## Function Keys

`F1`, `F2`, `F3`, ..., `F12`

## Arrow Keys

`LEFT`, `UP`, `RIGHT`, `DOWN`

## Number Keys

`0`, `1`, `2`, ..., `9`

## Punctuation Keys

- `COMMA` or `,`
- `PERIOD` or `.`
- `SEMICOLON` or `;`
- `SLASH` or `/`
- `BACKSLASH` or `\`
- `QUOTE` or `'`
- `BACKTICK` or `` ` ``
- `MINUS` or `-`
- `EQUALS` or `=`
- `LBRACKET` or `[`
- `RBRACKET` or `]`

## Letter Keys

`A`, `B`, `C`, ..., `Z` (case insensitive)
